How a Web Page Works

The web page request process.

Our 3 main technologies:

  • HTML. Used to define the different pieces of content in your webpage and their relevance.
  • CSS. Used to define your webpage's appearance.
  • JavaScript. Used to define your webpage's behavior.

HTML Concepts

HTML consists of a number of tags which define content. Tags are basically keywords with < and > around them.

<img> <-- this is a tag for defining images.

<p> <-- this is a tag for defining paragraph text.

You don't always use tags by themselves. Most tags have optional attributes, which attach important information to the tags.

Many tags like <p> consist of an opening and closing tag.for example, to define a paragraph we need both a starting and ending tag:

<p>This is the paragraph text.</p>

Other tags like <br> (for line breaks) are called self-closing; that is, they don't need closing tags.

This is one line.
This is another line. We don't need a closing tag.

Tags are often nested. You can put them inside each other where appropriate.

<p>This is the paragraph text,<br>with a line break inside it.</p>

Tags contain attributes, which alter their behavior and appearance.

<img src="my-image.png"> <-- "src" is an attribute of the <img> tag.

Essential HTML Tags

These are most of the HTML tags that are most important and common. There are more, but get good with these first.

<!doctype html>

Goes at the very top of the page. Defines the type of document.

For the most part you should never have to change this, just always include it.
<html>...</html> Defines the html document. Goes around everything except the <!doctype>
<head>...</head> Contains non-visible information about the webpage, such as the title, stylesheets, and description for search engines.
<title>...</title> Contains the title of the webpage that will show up in the browser.
<body>...</body> Contains all the visible content of the page, and more.
<h1>...</h1>,
<h2>...</h2>,
<h3>...</h3>,
<h4>...</h4>,
<h5>...</h5>,
<h6>...</h6>

These wrap headings for different sections of the page.

<h1> wraps the most important heading (usually the page's title). <h2> tags wrap sub-headings.

<h3> tags contain SUB-sub-headings.

...and so on.

Example:
<h1>My Website.</h1>
<h2>Nothing but great things here.</h2>
<p>...</p> These contain paragraph text.
Example:
<p>First paragraph.</p>
<p>Second paragraph.</p>
<img> Tags for displaying images. The "src" attribute is required to show anything.
Example:
<img src="my-image.jpg">
<a>...</a>

Anchor tags.

These are used for linking (or "hyperlinking") to another webpage, or to another part of the same page.

Examples:
<a href="http://attck.io">Join our mailing list!</a>
<a href="#about">Link to about section.</a>
<div>...</div> and <span>...</span>

Grouping Tags.

Both define sections of content in a webpage.

<div> tags define "block" sections.

<span> tags define sections within a line ("inline" sections).

Examples:

<div id="about">
<h2>About Our Company</h2>
<p>Bunch of text goes here. Inline elements may have <span>span</span> tags around them.</p>
</div>
<-- ... -->

Comment tags.

Anything between these tags is a "comment," or a note just for developers. It will not be seen in the web browser, only in the source code.

Examples:

<p>You can see this text in a browser.</p>
<!-- This text is invisible. -->

Standard HTML Page Structure

<!DOCTYPE html>
<html>
  <head>
    <title>
      title of the page here
    </title>
    <!-- Any stylesheet <link> tags go here. -->
  </head>
  <body>
    page contents here

    <!-- Any javascript <script> tags go here. -->
  </body>
</html>
