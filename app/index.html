How a Web Page Works

The web page request process.

Our 3 main technologies:

  • HTML. Used to define the different pieces of content in your webpage and their relevance.
  • CSS. Used to define your webpage's appearance.
  • JavaScript. Used to define your webpage's behavior.

HTML Concepts

HTML consists of a number of tags which define content. Tags are basically keywords with < and > around them.

<img> <-- this is a tag for defining images.

<p> <-- this is a tag for defining paragraph text.

You don't always use tags by themselves. Most tags have optional attributes, which attach important information to the tags.

Many tags like <p> consist of an opening and closing tag.for example, to define a paragraph we need both a starting and ending tag:

<p>This is the paragraph text.</p>

Other tags like <br> (for line breaks) are called self-closing; that is, they don't need closing tags.

This is one line.
This is another line. We don't need a closing tag.

Tags are often nested. You can put them inside each other where appropriate.

<p>This is the paragraph text,<br>with a line break inside it.</p>

Tags contain attributes, which alter their behavior and appearance.

<img src="my-image.png"> <-- "src" is an attribute of the <img> tag.

Important to note: HTML tags and attributes are case insensitive. That means uppercase and lowercase don't really matter - BUT keep everything lowercase unless there's a real reason not to.

So <body> is the same as <BODY> and <bOdY>. But you should always write it as <body> since that's the standard practice.

Essential HTML Tags

These are most of the HTML tags that are most important and common. There are more, but get good with these first.

<!doctype html>

Goes at the very top of the page. Defines the type of document.

For the most part you should never have to change this, just always include it.
<html>...</html> Defines the html document. Goes around everything except the <!doctype>
<head>...</head> Contains non-visible information about the webpage, such as the title, stylesheets, and description for search engines.
<title>...</title> Contains the title of the webpage that will show up in the browser.
<body>...</body> Contains all the visible content of the page, and more.
<h1>...</h1>,
<h2>...</h2>,
<h3>...</h3>,
<h4>...</h4>,
<h5>...</h5>,
<h6>...</h6>

These wrap headings for different sections of the page.

<h1> wraps the most important heading (usually the page's title). <h2> tags wrap sub-headings.

<h3> tags contain SUB-sub-headings.

...and so on.

Example:
<h1>My Website.</h1>
<h2>Nothing but great things here.</h2>
<p>...</p> These contain paragraph text.
Example:
<p>First paragraph.</p>
<p>Second paragraph.</p>
<img> Tags for displaying images. The "src" attribute is required to show anything.
Example:
<img src="my-image.jpg">
<a>...</a>

Anchor tags.

These are used for linking (or "hyperlinking") to another webpage, or to another part of the same page.

Examples:
<a href="http://attck.io">Join our mailing list!</a>
<a href="#about">Link to about section.</a>
<div>...</div> and <span>...</span>

Grouping Tags.

Both define sections of content in a webpage.

<div> tags define "block" sections.

<span> tags define sections within a line ("inline" sections).

Examples:

<div id="about">
<h2>About Our Company</h2>
<p>Bunch of text goes here. Inline elements may have <span>span</span> tags around them.</p>
</div>
<-- ... -->

Comment tags.

Anything between these tags is a "comment," or a note just for developers. It will not be seen in the web browser, only in the source code.

Examples:

<p>You can see this text in a browser.</p>
<!-- This text is invisible. -->

Common HTML Attributes

There are a few attributes that can be applied to almost any tag. There are 2 that you will see very often:

class

Assigns a css class to reference this element by.

You'll often use css classnames to style elements.

Classes can (and often do) apply to multiple elements at once.

Example:

<-- HTML code -->
<p class="blue-text">This text will be blue.</p>
<p class="blue-text">This text will also be blue.</p>
<p>This text will not.</p>

<-- CSS code -->
.blue-text { color: blue; }
id

Give an element an ID to refer to it directly.

IDs must be unique, just like any other ID - driver's license, etc.

These are often used with JavaScript, and sometimes with CSS.


<-- HTML code -->
<a id="my-link">Click me!</a>
<p id="my-text"></p>
<p>This text will not.</p>

<-- CSS code -->
.blue-text { color: blue; }

Standard HTML Page Structure

<!DOCTYPE html>
<html>
  <head>
    <title>
      title of the page here
    </title>
    <!-- Any stylesheet <link> tags go here. -->
  </head>
  <body>
    page contents here

    <!-- Any javascript <script> tags go here. -->
  </body>
</html>

CSS Concepts

CSS stands for Cascading Style Sheets. It's the programming language you use for changing the appearance of your webpage - styling, layout, color, fonts, etc. It can also do advanced things like animation.

CSS is made up of little pieces of code called rules.

A rule is made up of 2 main parts, a selector and a declaration, which is a list of style properties and values.

That's a lot of terms, but it's not really complicated. Here's an example:

The parts of a CSS rule.

What each piece does:

  • Selector: Targets which HTML element (or elements) we want to style. Think of it as a way to "find" the element(s).
  • Declaration: Tells us how to style the element(s). Each property and value change the elements' appearance somehow.

HTML elements have default display types which alter how some CSS properties affect them. There are a bunch, but the two types of display types you should be concerned with now are:

  • Inline. These are small elements that surround a piece of text that is part of something bigger.

    <span>, <img>, <strong> and <em> are examples of elements that are inline by default.

    Inline elements are not affected by CSS properties like width, height.

  • Block. Block elements can be affected by properties like width and height, and by default expand the full width of the page.

    <p>, <div> and the heading tags <h1>...<h6> are some elements that are block display by default.

Basic CSS Properties

color

Changes the color of the element. Duh.
font-family,
font-size,
font-style,
font-weight

These change various text properties of the element.

Examples:
h1 {
  font-family: "Helvetica";
  font-size: 40px;
  font-style: italic;
  font-weight: normal;
}

background-color,
background-image,
background-repeat,
background

These set the background of an element (obviously). "background" is a shortcut for all the others, and ones that aren't listed here.

border-style,
border-color,
border-width,
border

Borders go around an element, in between the margins and padding.

margin-top,
margin-right,
margin-bottom,
margin-left,
margin

Margins are the spacing around an element, beyond its border.

padding-top,
padding-right,
padding-bottom,
padding-left,
padding

Padding is the spacing around an element, inside its border.

JavaScript Concepts

JavaScript is used to change the behavior of a webpage.

Often, things that happen when you click, type, or otherwise interact with a webpage can be changed with JavaScript. It can also do things on its own in the background.

Basic things you must learn about JavaScript:

  • The DOM, or Document Object Model. The DOM is how JavaScript interacts with HTML elements.
  • Events. When writing JavaScript that responds to a person's behavior in the browser, you do them in response to events - things like mouse clicks, mouse movement, typing, and so on.
  • Variables are places to save information in your JavaScript code.

Variables in JavaScript

Variables are ways of saving information temporarily in a webpage. We define them using the var keyword. They can hold different types of information, such as numbers, strings (text), arrays (lists of values), and objects (lists of string/value pairs).

// "String" (text) variable.
var name = "Dave";

// Numeric variable.
var age = 25;

// "Array" variable.
var musicians = ["The Beatles", "Led Zepplin", "Taylor Swift"];

// "Object" variable.
var car_colors = {
  ferrari: "red",
  mercedes: "black",
  bmw: "white"
};

Important Tools & Techniques

View Source

See the HTML source of a webpage.

Click View -> Page Source. (This may differ in other browsers.)
Inspect Element

Look at a single HTML element on the page, and all information about it.

Right-click, then click Inspect Element. (This may differ in other browsers.)
JavaScript Console

In Chrome, go to View -> Developer -> JavaScript Console.

Type in basic JavaScript commands here, and run them by pressing Enter.

Also, when you encounter JavaScript errors, they will show up here in the console.
